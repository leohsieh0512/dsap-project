# 成績計算系統
## Proposal Report

### 動機與目標
<!-- 說明為什麼想做這個專題 -->
平常在計算考試成績時，常常需要自己算平均、找最高分和最低分，容易花很多時間。因此我想做一個簡單的成績計算系統，讓使用者可以輸入多筆成績，並快速得到統計結果。
### 預期功能

2.顯示所有成績

3.計算平均分數

4.顯示最高分

5.顯示最低分

## 競品比較
目前市面上已有許多成績計算工具，例如 Excel、線上 GPA 計算器等，這些工具通常功能完整，但操作上可能較複雜，或需要使用者自行整理資料。

本專題與現有工具的差異如下：

1.支援 CSV 成績匯入:使用者可以先在 Excel 整理成績，再匯出為 CSV 檔，由系統一次匯入所有資料，避免逐筆輸入。
2.提供簡單互動式回饋:系統會根據成績自動提示，例如提醒需要加強的科目，或對高分給予鼓勵，使結果不只是數值顯示。
