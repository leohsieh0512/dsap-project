# 成績計算系統
## Proposal Report

### 動機與目標
<!-- 說明為什麼想做這個專題 -->

## 競品比較
目前市面上已有許多成績計算工具，例如 Excel、線上 GPA 計算器等，這些工具通常功能完整，但操作上可能較複雜，或需要使用者自行整理資料。

本專題與現有工具的差異如下：

1.支援 CSV 成績匯入:使用者可以先在 Excel 整理成績，再匯出為 CSV 檔，由系統一次匯入所有資料，避免逐筆輸入。

2.提供簡單互動式回饋:系統會根據成績自動提示，例如提醒需要加強的科目，或對高分給予鼓勵，使結果不只是數值顯示。

## Final Report

### 專案說明
<!-- 完整描述你的專案做了什麼 -->
本專案為一個成績計算系統，使用 Python 撰寫，提供基本的成績管理與分析功能。使用者可以手動輸入或匯入 CSV 檔案加入成績資料，系統會自動進行計算與分析。系統主要功能包括成績顯示、平均分數計算、最高分與最低分查詢，以及簡單的學習回饋。此外，本系統也支援匯入舊成績資料，進行進步與退步分析，幫助使用者了解學習狀況的變化。在演算法方面，本專案比較了兩種尋找最高分的方法（for 迴圈與內建 max 函式），並進行簡單的效能比較，藉此探討不同實作方式的差異。

### 使用方式
<!-- 如何編譯、執行、使用你的程式 -->
執行程式後，會看到一個選單，可以依照需求選擇不同功能，可以先準備一個 CSV 檔，再透過系統一次匯入所有成績。匯入完成後，可以查看所有成績，並計算平均分數、最高分和最低分。系統也會根據成績提供簡單的回饋，例如提醒是否有需要加強的科目。整體操作只需要依照畫面提示輸入對應的數字即可完成。
