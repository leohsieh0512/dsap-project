# 成績計算系統
## Proposal Report

### 動機與目標
<!-- 說明為什麼想做這個專題 -->
平常在計算考試成績時，常常需要自己算平均、找最高分和最低分，容易花很多時間。因此我想做一個簡單的成績計算系統，讓使用者可以輸入多筆成績，並快速得到統計結果。
### 預期功能
<!-- 列出預計實作的功能 -->
1.新增成績

2.顯示所有成績

3.計算平均分數

4.顯示最高分

5.顯示最低分
### 使用技術
<!-- 使用的語言、框架、工具等 -->
1.Python

2.list 資料結構

3.while 迴圈

4.if 判斷式

5.sum、max、min 函式
### 時程規劃
<!-- 各週預計完成的進度 -->
1.Week 7：決定題目與規劃功能

2.Week 9：完成新增成績與顯示成績功能

3.Week 10：完成平均分數、最高分、最低分功能

4.Week 11：完成 Proposal Report

5.Week 14：完成 Final Report 與 Demo 影片
### 與課程的關聯
<!-- 你的專題可能涉及哪些資料結構或演算法概念？為什麼？ -->
